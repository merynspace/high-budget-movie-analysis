## Project Overview
This project analyzes financial and performance trends among high-budget movies using SQL and Power BI. The dashboard explores revenue, profitability, release season trends, genre performance, and return on investment (ROI).

The goal of the project was to practice:
- SQL data cleaning and transformation
- Data analysis
- Power BI dashboard development
- Data storytelling and visualization

---

## Tools Used
- MySQL
- Power BI
- Excel

---

## Dataset
Source:
Mitchell Harrison's Top 500 Movies by Production Budget dataset

The dataset included:
- movie titles
- release dates
- production budgets
- worldwide revenue
- genres
- runtimes

---

## Data Cleaning & Transformation
Performed in MySQL:
- handled missing values
- replaced zero values
- created profit column
- created ROI calculations
- created release season categories

Example SQL concepts used:
- CASE statements
- GROUP BY
- Aggregate functions
- Date functions
- Calculated columns

---

## Key Insights
- Action and Adventure films generated the highest total revenue.
- Spring and Summer releases produced the highest profits.
- Average ROI exceeded median ROI, indicating blockbuster outliers.
- Longer runtimes did not consistently guarantee higher profitability.
- Revenue declined sharply around 2020, likely due to pandemic-related theater closures.
